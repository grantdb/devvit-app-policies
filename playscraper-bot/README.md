# Playscraper Bot
PlayScraper Bot automatically detects, extracts, and summarizes application data linked in your community.

## Overview
The Playscraper Bot automatically detects, extracts, and summarizes application data linked in your community. Whether users share a Google Play Store app or an open-source GitHub repository, this bot instantly provides the community with a clean, standardized metadata summary.

## How It Works
When a user posts a comment containing a supported link, the bot immediately fetches the relevant metadata—such as the app description, update date, developer name, and repository stars—and formats it into a highly readable mod comment.

## Features
- **Multi-Platform Support:** Seamlessly detects and processes both Google Play Store links and GitHub repository URLs.
- **Clean Formatting:** Standardizes application details into a consistent layout, making it easy for community members to digest app information at a glance.
- **Fully Automated:** Requires zero manual intervention after setup. It actively monitors submissions and applies metadata context intelligently.
