Playscraper Bot is a professional moderator utility that leverages advanced AI to extract and summarize metadata from application links. By providing standardized, informative headers for mobile applications and open-source projects, it empowers moderators to verify and present shared content with surgical precision.

## Core Features

- **Surgical Metadata Extraction**: Expertly identifies and processes deep links from the Google Play Store and GitHub repositories.
- **Generative AI Intelligence**: Powered by **Google Gemini 2.5 Flash** for high-speed, high-accuracy software summarization.
- **Moderator Controlled**: Designed primarily as a moderator-triggered tool, ensuring that AI-generated summaries only appear when and where they are needed.
- **Modqueue Integration**: Automatically detects post approvals to ensure summaries are only generated for content that meets community standards.
- **Standardized Presentation**: Consolidates ratings, categories, licensing, and repository metrics into a single, cohesive community comment.

## Usage & Configuration

| Feature | Functionality |
|---------|-------------|
| **Manual Trigger** | Use the **"Trigger App Scraper"** menu item on any post or comment to run an on-demand scan. |
| **Gemini Integration** | Utilizes **Google Gemini 2.5 Flash** for industry-leading speed and extraction intelligence. |
| **Moderation Queue** | Intelligently waits for moderator approval on filtered posts before processing summaries. |
| **Smart Retries** | Robust architecture designed to handle Reddit rate limits and API transients gracefully. |
