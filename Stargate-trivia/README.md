# Stargate SG-1 Trivia Game
Stargate Trivia game with leaderboard and many curated canonical questions!

## Overview
The Stargate SG-1 Trivia Game brings the universe of the classic sci-fi series directly to your subreddit. It challenges your community with canonical trivia questions related exclusively to Stargate SG-1, its lore, and its cast.

## How It Works
Once installed, moderation teams can initiate interactive trivia events. The app handles posting the questions, rendering the gameplay UI smoothly within the Reddit client, and evaluating participant answers without requiring any manual tracking.

## Features
- **Curated Question Pool:** Features a heavily curated list of unique Stargate SG-1 questions designed to challenge your community.
- **Responsive Interface:** Custom-built interactive UI that resizes dynamically to fit viewport dimensions, ensuring players experience a seamless display.
- **Automated Scorekeeping:** Automatically processes responses and maintains game state entirely within the Reddit environment.
